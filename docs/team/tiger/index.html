# Code of Conduct

새로운 분에게 배울 자세가 되어 있는 TEAM SUNSIN.SHOP

  1. 행동강령(CoC) 위원회 작업 중 ...
  2. 참조 : 파이콘 행동 강령(https://2023.pycon.kr/coc/purpose)
  3. 우리는 함께 성장합니다.
  4. 모르는 것은 죄가 아니다. 모르는 것을 감추는 순간 큰 장애로 되돌아 온다.
  5. 프로젝트는 실패해도 팀은 남는다.
  6. 테스트 할 수 없는 것은 만들 수 없다.
  7. 애자일 선언 이면의 원칙 - 우리의 최우선 순위는, 가치 있는 소프트웨어를 일찍 그리고 지속적으로 전달해서 고객을 만족시키는 것이다, 작동하는 소프트웨어를 자주 전달하라 등등 ... - 을 위해 우리는 개발 극 초기에 PR 을 생성한다.
